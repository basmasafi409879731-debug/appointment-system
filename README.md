# نظام حجز مواعيد صالون التجميل (Beauty Salon Appointment System) 🌸

## 💡 فكرة المشروع
نظام إلكتروني لإدارة حجوزات صالون التجميل، يهدف إلى رقمنة عملية حجز المواعيد، مما يتيح للزبونات اختيار الخدمات المتاحة وتحديد التاريخ والوقت المناسب، مع توفير واجهة إدارة للأدمن لمتابعة الحجوزات.

## 📝 المشكلة التي يحلها
* **التنظيم:** التغلب على فوضى الحجوزات الورقية والاتصالات الهاتفية المتكررة.
* **الشفافية:** إتاحة فرصة للزبونات للاطلاع على قائمة الخدمات والأسعار بوضوح.
* **الكفاءة:** توفير وقت الأدمن في تلقي الحجوزات يدوياً وتقليل الأخطاء البشرية وتداخل المواعيد.

## 🛠️ التقنيات المستخدمة
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5.
* **Backend:** PHP.
* **Database:** MySQL.

## 📊 شرح الجداول (Database Schema)
* **`users`:** يخزن بيانات المستخدمين (اسم المستخدم، كلمة المرور المشفرة، والصلاحيات `admin/user`).
* **`services`:** يحتوي على قائمة الخدمات المقدمة (الاسم، السعر، ومسار الصورة).
* **`employees`:** يحتوي على أسماء الموظفين في الصالون.
* **`appointments`:** الجدول الرئيسي الذي يربط المستخدم بالخدمة والموظف والتاريخ وحالة الحجز (Pending, Confirmed, Cancelled).

## 🖥️ شرح الشاشات ولوحة التحكم
* **الصفحة الرئيسية:** تعرض الخدمات المتاحة مع نموذج للحجز السريع.
* **صفحة الحسابات:** تسجيل دخول جديد أو الوصول لحسابكِ الخاص.
* **صفحة مواعيدي:** واجهة مخصصة للزبونة لمراجعة حالة حجوزاتها.
* **لوحة تحكم الأدمن:** تتيح للأدمن استعراض جميع الحجوزات وتحديث حالتها.
